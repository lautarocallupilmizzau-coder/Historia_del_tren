📘 Historia del Tren – Proyecto HTML & CSS (NF1)
Este proyecto es una página web informativa creada como parte del módulo NF1. Su objetivo es presentar la historia del tren mediante una estructura HTML clara, estilos CSS básicos y una correcta organización de archivos. El proyecto demuestra dominio de conceptos fundamentales del desarrollo web y buenas prácticas en la gestión de rutas, imágenes y estructura de carpetas.
🚂 Descripción del proyecto
La página web ofrece una introducción a la evolución del ferrocarril, acompañada de imágenes, listas, tablas y navegación interna. El proyecto fue desarrollado desde cero, aplicando:

--> Estructura semántica en HTML

--> Estilos CSS para mejorar la presentación

--> Rutas relativas para cargar imágenes locales

--> Organización de archivos en un entorno de trabajo real

--> Resolución de errores comunes en editores como Phoenix Code

Este proyecto representa uno de los primeros pasos en mi camino como desarrollador web, mostrando mi capacidad para construir interfaces simples, funcionales y bien estructuradas.
🧩 Tecnologías utilizadas
--> HTML5

--> CSS3

--> Gestión de archivos y rutas relativas

--> Editor Phoenix Code

🎯 Objetivos de aprendizaje alcanzados
1) Aplicar estilos mediante clases e identificadores.
2) Insertar imágenes locales usando rutas relativas.
3) Crear listas ordenadas y desordenadas.
4) Construir tablas con encabezados y filas de datos.
5) Organizar un proyecto web en carpetas reales.
6) Publicar un proyecto en GitHub como parte de un portafolio profesional.
7) Comprender cómo funciona la estructura base de una página web.
🌐 Características principales
* Diseño simple y centrado en la lectura.
* Imágenes integradas correctamente mediante rutas relativas.
* Navegación interna fluida.
* Código limpio y fácil de entender.
* Proyecto ideal como base para futuros desarrollos más avanzados.
